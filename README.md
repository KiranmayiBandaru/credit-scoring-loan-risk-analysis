# credit-scoring-loan-risk-analysis

With the increasing number of loan applications, banks face the challenge of accurately assessing whether a customer can repay the loan without defaulting. This involves significant risk for banks, as lending money to customers with poor repayment history can lead to financial losses. Traditionally, banks evaluate loan applications manually by reviewing income documents, employment history, past loan records and CIBIL scores. However, this manual process is time consuming, requires significant human resources and is prone to bias and inconsistency. To address these challenges, this project proposes an automated loan risk analysis system that tracks customer behaviour based on their financial history and categorizes them into risk levels to determine loan eligibility. The system employs machine learning algorithms including Logistic Regression, Random Forest and XGBoost to analyze credit scoring data and predict whether a customer is eligible for loan sanction with high accuracy and minimal error.
